Acerca de esta aplicación

Bienvenido/a a esta app de gestión de tareas, creada como práctica de Angular.
  • Routing entre varias páginas
  • Formulario reactivo con validación (crear y eliminar tareas)
  • Servicios e inyección de dependencias
Esta aplicación permite:
  • Agregar nuevas tareas con título, descripción y estado de completada
  • Eliminar tareas existentes
  • Navegar fácilmente entre páginas con la barra de navegación
